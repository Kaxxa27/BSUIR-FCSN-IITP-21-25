# Решение задачи №8: Одноканальная СМО с отказами и ремонтом

## Условие задачи
Рассматривается **одноканальная СМО с отказами**. Входящий поток заявок поступает с интенсивностью \( X \). Время обслуживания имеет показатель распределения с параметром \( Y = \frac{1}{t_{обс}} \). Канал может время от времени выходить из строя (отказы), при этом интенсивность потока отказов — **простая с интенсивностью \( X \)**. Время восстановления (ремонта) канала описывается показательным распределением с параметром \( R = \frac{1}{t_{рем}} \).

Состояния системы:
- **s0** — канал свободен;
- **s1** — канал занят (обслуживается заявка, канал исправен);
- **s2** — канал ремонтируется.

Требуется найти **финальные вероятности состояний СМО** и основные характеристики.

---

## Построение модели системы

Одноканальная СМО с отказами и ремонтом можно рассматривать как **марковский процесс** с тремя состояниями:

| Обозначение | Описание состояния       |
|-------------|-------------------------|
| **s0**      | Канал свободен           |
| **s1**      | Канал занят заявкой      |
| **s2**      | Канал ремонтируется      |

### Граф переходов

На основании условия задачи строим **граф переходов между состояниями**:

```plaintext
        λ            μ           γ
   s0 -------> s1 --------> s2 --------> s0
```
где:
- \( \lambda = X \) — интенсивность поступления заявок;
- \( \mu = Y \) — интенсивность обслуживания;
- \( \gamma = R \) — интенсивность ремонта канала.

### Дифференциальные уравнения Колмогорова

Для описания вероятностей нахождения системы в состояниях \( s0, s1, s2 \), введем **переменные**:
- \( p_0 \) — вероятность нахождения системы в состоянии **s0**;
- \( p_1 \) — вероятность нахождения системы в состоянии **s1**;
- \( p_2 \) — вероятность нахождения системы в состоянии **s2**.

Тогда система уравнений Колмогорова имеет вид:

\[
\begin{cases}
\frac{dp_0}{dt} = -\lambda p_0 + \gamma p_2, \\
\frac{dp_1}{dt} = \lambda p_0 - \mu p_1, \\
\frac{dp_2}{dt} = \mu p_1 - \gamma p_2.
\end{cases}
\]

В **стационарном режиме** производные равны нулю, то есть система достигает **равновесного состояния**:

\[
\begin{cases}
-\lambda p_0 + \gamma p_2 = 0, \\
\lambda p_0 - \mu p_1 = 0, \\
\mu p_1 - \gamma p_2 = 0.
\end{cases}
\]

К уравнениям добавляем **нормировочное условие**:
\[
p_0 + p_1 + p_2 = 1.
\]

---

## Решение системы уравнений

Решаем систему уравнений:

1. Из первого уравнения:
\[
p_2 = \frac{\lambda}{\gamma} p_0.
\]

2. Из второго уравнения:
\[
p_1 = \frac{\lambda}{\mu} p_0.
\]

3. Подставляем выражения для \( p_1 \) и \( p_2 \) в нормировочное условие:
\[
p_0 + \frac{\lambda}{\mu} p_0 + \frac{\lambda}{\gamma} p_0 = 1.
\]

Выносим \( p_0 \) за скобки:
\[
p_0 \left( 1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma} \right) = 1.
\]

Находим \( p_0 \):
\[
p_0 = \frac{1}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}}.
\]

4. Найдем \( p_1 \) и \( p_2 \):
\[
p_1 = \frac{\lambda}{\mu} p_0 = \frac{\frac{\lambda}{\mu}}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}},
\]
\[
p_2 = \frac{\lambda}{\gamma} p_0 = \frac{\frac{\lambda}{\gamma}}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}}.
\]

---

## Финальные вероятности состояний

| Состояние | Вероятность \( p \)                                       |
|-----------|---------------------------------------------------------|
| **s0**    | \( p_0 = \frac{1}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}} \) |
| **s1**    | \( p_1 = \frac{\frac{\lambda}{\mu}}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}} \) |
| **s2**    | \( p_2 = \frac{\frac{\lambda}{\gamma}}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}} \) |

---

## Основные характеристики системы

1. **Вероятность отказа** (канал занят или ремонтируется):
   \[
   P_{отказ} = p_1 + p_2 = 1 - p_0.
   \]

2. **Среднее время восстановления канала**:
   \[
   T_{рем} = \frac{1}{\gamma}.
   \]

3. **Нагрузка на канал**:
   \[
   \rho = p_1.
   \]

4. **Относительная пропускная способность**:
   \[
   Q = 1 - P_{отказ} = p_0.
   \]

5. **Абсолютная пропускная способность**:
   \[
   A = \lambda Q = \lambda p_0 = \frac{\lambda}{1 + \frac{\lambda}{\mu} + \frac{\lambda}{\gamma}}.
   \]
   
---

## Заключение

Таким образом, мы нашли **стационарные вероятности состояний СМО** и основные характеристики системы. Полученные результаты позволяют проанализировать поведение системы при различных значениях параметров \( \lambda, \mu, \gamma \).
